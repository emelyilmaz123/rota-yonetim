# Rota Yonetim - Otobus Firmasi Yonetim Sistemi

Antalya-Isparta-Afyon-Konya-Ankara guzergahinda faaliyet gosteren otobus firmasi icin gelistirilmis web tabanli yonetim sistemi.

## Ozellikler
- Yolcu kayitlari ve koltuk numarasi takibi
- Binis/inis noktalari ve ucret yonetimi
- Guzergah uzerindeki mola noktalari yonetimi
- Etkinlik modulu
- Vercel Postgres ile canli veritabani
